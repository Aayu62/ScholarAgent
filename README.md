# ScholarAgent

ScholarAgent is a multi-agent AI research assistant powered by RAG, LangChain, CrewAI, FAISS, and Gemini APIs.

## Features
- PDF document analysis
- Semantic search
- Multi-agent workflows
- AI-powered research assistance
- Report generation

## Tech Stack
- FastAPI
- Streamlit
- LangChain
- CrewAI
- FAISS
- Gemini API
